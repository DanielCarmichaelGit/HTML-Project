What are protocols and how do they work?

Protocols are how you interact with your data. Post, Get, Push, or Delete data using our easy to read endpoints, thats a protocol

How do I integrate Mongified?

Download mongified using npm i mongified, copy your Mongified handle and set it as the database endpoint for this mongified project. Start building data!

How can I manage my database?

Download our mobile app and see non user specific data, or continue to your dashboard to see it all (some data is encrypted to protect the end user)
